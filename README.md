# Market Automation

This project is a market automation desktop application developed with C#, Windows Forms, ADO.NET, and SQL Server.

## Project Features
- Product management
- Customer management
- Category management
- User login system
- Sales transaction operations
- Panel-based management screens
- Database connection with SQL Server

## Technologies Used
- C#
- Windows Forms
- ADO.NET
- SQL Server

## Purpose
The purpose of this project is to manage basic market operations in a desktop environment and improve database-based application development skills.

## Note
This project was developed for educational purposes.
